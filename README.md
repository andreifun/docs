# Strawberry Nebula Docs

Documentation source for Strawberry Nebula, built with Mintlify.

## Local development

1. Install Mintlify CLI:

```bash
npm i -g mint
```

2. Run the local preview from the repo root:

```bash
mint dev
```

3. Open `http://localhost:3000`.

## Quality checks

Check for broken links before opening a pull request:

```bash
mint broken-links
```

## Publishing

Changes deploy automatically when updates are pushed to the default branch.
