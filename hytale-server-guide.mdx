Welcome to Team0 Nebula, here you can publish your Hytale server onto out public server list, get new players, manage a newsletter, manage marketing and so much more.

## Before we start you'll need:

- A Hytale server. You can see how to start a server [here](/how-to-start-a-server).
- A Team0 Nebula Account. Log in here. (TODO: ADD LOG IN LINK)

<Note>
  In order to show your server's status you need to install [HyQuery](https://www.curseforge.com/hytale/mods/hyquery) first.
</Note>

## Setting up the server page.

<CreateServer />

Then complete these fields:

- Enter your server's display name. This is the name shown in the server listing.
- Select **Hytale** in the <Icon icon="gamepad" /> **Game** dropdown.
- Enter your server address and game version.
- Choose a <Tooltip tip="The slug is the text that appears in your server page URL. Think of it like a social media handle.">slug</Tooltip> for your page URL.
